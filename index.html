dffptch.js

A micro library for diffing and patching JSON objects using a compact diff format

What

Why

If your client is sending a lot of updates to your backend – as it might in a collaborative app or real-time game – transfering the entire changed JSON wastes a lot of bandwitdh. dffptch.js makes sending only the changes in a compact format very easy.
